# My Portfolio

Personal portfolio site built with Next.js 16 + React 19.

## Stack

- Next.js 16 / React 19
- TypeScript
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Customize

All personal data lives in `lib/data.ts` — projects, skills, and experiences. Just edit that file to make it yours.

## Structure

```
app/           → pages and layouts
components/    → UI components (Bio, About, Projects, Skills, Experiences, etc.)
lib/           → data and utilities
public/        → static assets
```
